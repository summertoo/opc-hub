# 企业确权审核规范

## 目的

确保 OPC Hub 目录中的每一位成员都是真实经营的一人公司/超级个体，维护目录的可信度和质量。

## 审核委员会

由项目创建者及联合创始人组成，负责所有 PR 的最终审核。

## 申请人需提供

提交 PR 时，在 PR 描述中附上以下任意一项证明材料：

| 证明类型 | 说明 |
|---------|------|
| 营业执照 | 个体工商户或一人有限责任公司营业执照截图（可打码敏感信息） |
| 平台主页 | 独立网站、GitHub 主页、即刻/领英等专业平台主页链接 |
| 作品集 | 可公开访问的代表作品或案例链接 |
| 社交证明 | 有一定影响力的社交媒体账号（粉丝数 > 500） |

## 审核标准

- ✅ 真实存在且在运营中
- ✅ 档案信息与证明材料一致
- ✅ 技能描述具体、可信
- ❌ 虚假信息或无法核实
- ❌ 纯广告/营销内容
- ❌ 重复注册

## 审核流程

1. 申请人提交 PR
2. 审核委员会在 **7个工作日** 内完成审核
3. 审核通过 → 合并 PR，手动重建索引
4. 审核不通过 → 在 PR 中说明原因，申请人可修改后重新提交

## 档案更新

已入驻成员更新自己的档案，只需提交新 PR，审核委员会快速复核即可。

## 退出机制

如需从目录中移除，提交 PR 删除对应文件，或联系审核委员会处理。
